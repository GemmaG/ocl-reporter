Unicode text normalization for OCaml

Uunf is an OCaml module for normalizing Unicode text. It supports all Unicode [normalization forms][1] and is independent from any IO mechanism or Unicode text data structure. Text can be processed without a complete in-memory representation.

Uunf is made of a single independent module and distributed under the BSD3 license.

[1]: http://www.unicode.org/reports/tr15/
